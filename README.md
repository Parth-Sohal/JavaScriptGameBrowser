# 🎮 JavaScript Arcade Hub

A collection of interactive browser-based mini games built using **Vanilla JavaScript, HTML, and CSS**.  
The platform acts as a **central arcade hub** where users can explore and play multiple games from a single interface.

🔗 **Live Demo:** https://javascript-arcade-hub.vercel.app

---

## 🚀 Features

- Multiple browser games in one platform
- Interactive and dynamic UI
- Real-time game logic using JavaScript
- DOM manipulation and event handling
- Simple and responsive interface
- Easy navigation between games

---

## 🎯 Games Included

### 🎲 Guess The Number
Players try to guess a randomly generated number with hints for higher or lower guesses.

### ❌⭕ Tic Tac Toe
A classic two-player game with win detection and reset functionality.

### 🎲 Dice Game
Simulates dice rolls and determines a winner dynamically.

### 🚗 Car Finish Game
A simple racing-style game where the player controls a car to reach the finish line.

## 🛠 Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
